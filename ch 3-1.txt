LIST OF CASE MANIPULATION FUNCTIONS : 1 UPPER 2 LOWER 3 INITCAP

UPPER: IT WILL CONVERT THE CHARACTER IN UPPER CASE

LOWER: IT WILL CONVERT IN LOWER CASE 

INTICAP: IT WILL CONVERT IN CAMEL CASES

SELECT FIRST_NAME,UPPER(FIRST_NAME) FROM EMPLOYEES;

SELECT FIRST_NAME,LOWER(FIRST_NAME) FROM EMPLOYEES;

SELECT INITCAP('AmaR akBAr ANthoNY') FROM DUAL;

SELECT INITCAP('AmaR_akBAr#ANthoNY') FROM DUAL;

 SPECIAL CHARACTERS ARE CONSIDERED AS WORD SEPARATOR. BUT DIGITS ARE NOT CONSIDERED AS WORD SEPARATOR.

 SELECT INITCAP('AmaR5akBAr7ANthoNY') FROM DUAL;

 SELECT INITCAP('AmaR5akBAr^ANthoNY') FROM DUAL;
